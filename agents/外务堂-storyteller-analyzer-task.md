## 任务目标
分析 X 新闻，生成通俗易懂的报告。

## 输入
- 原始采集: ~/Documents/long.zhu/Inbox/YYYY-MM-DD/评书先生-X/YYYY-MM-DD/X-news_HH-MM.md

## 输出
- 分析报告: ~/Documents/long.zhu/Inbox/YYYY-MM-DD/评书先生-X/YYYY-MM-DD_HH-MM.md
- 文件大小: > 4KB

## 分析流程

```
1. 读取 X-News 原始采集
2. 筛选 Top 3 高分新闻 (>=75)
3. 使用 MiniMax-M2.1 生成摘要 (≥200字)
4. 使用 MiniMax-M2.1 生成点评 (≥200字)
5. 保存报告
```

## 质量要求（墨刑审核标准）

### 必须包含
- **中文摘要** (每条新闻): 200字以上
- **中文点评** (每条新闻): 200字以上
- **趋势总结**: 500字以上

## 审核流程

1. 评书先生完成分析 → 保存报告
2. 提交内务堂（墨刑）审核质量
3. 墨刑审核通过 → 归档到 OB 仓库
4. 归档路径: `~/Documents/long.zhu/Inbox/YYYY-MM-DD/评书先生-X/`
