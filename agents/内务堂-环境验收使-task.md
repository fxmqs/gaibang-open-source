## 验收标准

| # | 检查项 | 要求 | 检查方式 |
|---|--------|------|----------|
| 1 | 安装日志 | 软件版本符合要求 | 读取日志 |

| 3 | 自测通过 | 自检脚本通过 | 执行自检 |
| 4 | 磁盘空间 | >10GB 可用 | df 命令 |

## 验收流程

### Step 1: 接收验收任务
```bash
# 读取验收信息
- 令牌ID: LINGPai_20260215_MOBILE
- 任务: Android+iOS 环境安装
- 报告: Inbox/2026-02-15/Mobile/mobile_env_2026-02-15.md
```

### Step 2: 逐项验收
```bash
# 1. 检查安装日志
if grep -q "Android Studio" "$REPORT" && \
   grep -q "Xcode" "$REPORT"; then
  echo "✅ 关键软件已安装"
else
  echo "❌ 关键软件缺失"
fi

# 2. 检查环境可用
if command -v java > /dev/null 2>&1; then
  JAVA_VER=$(java -version 2>&1 | head -1)
  echo "✅ Java: $JAVA_VER"
else
  echo "❌ Java 未安装"
fi

# 3. 检查自检结果
if grep -q "成功率: 100%" "$REPORT"; then
  echo "✅ 自检通过"
else
  echo "⚠️ 自检有问题"
fi

# 4. 检查磁盘空间
FREE_SPACE=$(df -h / | tail -1 | awk '{print $4}')
echo "📊 磁盘空间: $FREE_SPACE"
```
