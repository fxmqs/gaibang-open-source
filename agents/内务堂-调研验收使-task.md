## 验收标准

## 验收流程

### Step 1: 接收验收任务
```bash
# 读取验收信息
- 令牌ID: LINGPai_20260215_RESEARCH
- 弟子: 调研监控使
- 任务: Moltbook 行业调研
- 报告: Inbox/2026-02-14/帮主/调研/xxx.md
```

### Step 2: 逐项验收
```bash
# 1. 检查报告结构
if grep -q "# " "$file" && \
   grep -q "## " "$file"; then
  echo "✅ 报告结构完整"
else
  echo "❌ 报告结构不完整"
fi

# 2. 检查分析深度
CONTENT_LEN=$(wc -c < "$file")
if [ "$CONTENT_LEN" -gt 5000 ]; then
  ANALYSIS_LEN=$(grep -o "分析" "$file" | wc -l)
  echo "✅ 分析深度: 约 $CONTENT_LEN 字"
else
  echo "⚠️ 分析内容较少"
fi

# 3. 检查建议可行性
if grep -q "建议" "$file" && \
   grep -q "执行" "$file"; then
  echo "✅ 建议具有可行性"
else
  echo "⚠️ 建议可能不够具体"
fi

# 4. 检查丐帮签名
if grep -q "🏮 帮主-.*汇报：" "$file" && \
   grep -q ".*告退 ✅"; then
  echo "✅ 丐帮签名完整"
else
  echo "❌ 丐帮签名缺失"
fi
```
