## 验收标准

| # | 检查项 | 要求 | 检查方式 |
|---|--------|------|----------|
| 1 | Front Matter | tags/created/author/source 完整 | 读取文件 |
| 2 | 丐帮签名 | 开头+结尾 | 搜索签名 |

| 4 | 摘要 | >=200字/条 | 读取摘要 |
| 5 | 点评 | >=200字/条 | 读取点评 |

## 验收流程

### Step 1: 接收验收任务
```bash
# 读取验收信息
- 令牌ID: LINGPai_20260215_XXXX
- 产出文件: Inbox/2026-02-15/外务堂/模型侠/hf_models_2026-02-15_08-08.md
- 任务要求: >= 10 条模型数据
```

### Step 2: 逐项验收
```bash
# 1. 检查 Front Matter
if grep -q "tags:" "$file" && \
   grep -q "created:" "$file" && \
   grep -q "author:" "$file" && \
   grep -q "source:" "$file"; then
  echo "✅ Front Matter 完整"
else
  echo "❌ Front Matter 不完整"
fi

# 2. 检查丐帮签名
if grep -q "🏮 外务堂-.*汇报：" "$file" && \
   grep -q ".*告退 ✅"; then
  echo "✅ 丐帮签名完整"
else
  echo "❌ 丐帮签名缺失"
fi

# 3. 统计数据量
model_count=$(grep -c "^- " "$file" || echo "0")
echo "📊 数据量: $model_count 条"

# 4. 检查摘要（每条数据后跟摘要）
# 5. 检查点评（每条数据后跟点评）
```

### Step 3: 记录验收结果
