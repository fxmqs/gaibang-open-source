# 🐉 丐帮 - AI Agent 协作系统

> 开源的 AI Agent 协作与任务调度框架

## 📖 项目简介

丐帮是一个基于 OpenClaw 的 AI Agent 协作系统，采用武侠江湖风格的任务调度与验收机制。

### 核心特性

- 🎯 **任务令牌系统** - 基于令牌的任务调度
- 🛡️ **多层验收机制** - 6种验收使确保产出质量
- 🔄 **自动化流程** - 定时任务自动触发与验收
- 📊 **完整日志追踪** - 所有操作可追溯

## 🚀 快速开始

### 1. 创建新令牌

```bash
# 复制令牌模板
cp gai-bang/令牌筒/templates/令牌模板-数据采集.md 新令牌.md

# 编辑令牌内容
vi 新令牌.md

# 添加到令牌筒
cat 新令牌.md >> gai-bang/令牌筒/当前令牌筒.md
```

### 2. 令牌状态流转

```
🔴 待接令 → 🔴 执行中 → 🔴 待验收 → ✅ 完成
```

### 3. 触发验收

根据任务类型自动触发对应验收使：

| 任务类型 | 验收使 |
|----------|--------|
| 数据采集 | 数据验收使 |
| 技能开发 | 代码验收使 |
| 环境安装 | 环境验收使 |
| 整改修复 | 整改验收使 |
| 调研分析 | 调研验收使 |
| 系统运维 | 运维验收使 |
